# Learning Routing — [Profession] Pack

## Route 1: Behavioral Rule
**Target:** CLAUDE.md → [section name]
**Criteria:** Universal directive. Applies across all sessions. Imperative voice.
**Format:** Single bullet point with optional `_Why:` suffix.
**Examples:**
- "Always [action] before [action]"
- "Never [action] without [check]"

## Route 2: Specific Mistake
**Target:** .claude/lessons.md
**Criteria:** Tied to an incident. Has root cause. Pattern: "did X → should have done Y"
**Format:** `[DATE] [INSTINCT:0.30] CATEGORY: What happened → What to do instead`
**Categories:** [Define your categories here]

## Route 3: Methodology Insight
**Target:** Manual review (present to user)
**Criteria:** Strategic. Needs judgment. About approach, market, or methodology.
**Action:** Present options, don't auto-write.
