# Pipedrive Note Templates
# Load when creating/updating CRM notes. Referenced by gates.md.

## Standard Header Block (every note starts with this)
```html
<h3>CONTACT</h3>
<table>
<tr><td><b>Name</b></td><td>[Full Name]</td></tr>
<tr><td><b>Title</b></td><td>[Title]</td></tr>
<tr><td><b>Email</b></td><td>[email]</td></tr>
<tr><td><b>Phone</b></td><td>[phone]</td></tr>
<tr><td><b>LinkedIn</b></td><td>[URL]</td></tr>
<tr><td><b>Company</b></td><td>[Company Name]</td></tr>
<tr><td><b>Company Size</b></td><td>[X employees]</td></tr>
<tr><td><b>Industry</b></td><td>[industry]</td></tr>
<tr><td><b>Website</b></td><td>[URL]</td></tr>
<tr><td><b>Source</b></td><td>[Apollo, LinkedIn, etc.]</td></tr>
</table>
```

## Template 1: Pre-Demo Research Note
[Standard Header Block]
ICP FIT: Score X/10 — justification
COMPANY OVERVIEW: 2-3 sentences
PRIOR TOUCHPOINTS: Date — Type — Outcome
PAIN POINTS: numbered list
DEMO ANGLES: Lead with / Follow with / Skip
OBJECTIONS TO EXPECT: Objection → Response
CLOSE STRATEGY: offer/trial/pricing

## Template 2: Demo Summary Note
[Standard Header Block]
DEMO DETAILS: date, attendees, duration, Fireflies link
THREADS SHOWN: name + reaction/quote
BUY SIGNALS: specific signal with quote
OBJECTIONS RAISED: objection + handling + resolved Y/N
PRICING DISCUSSED: what + response
NEXT STEPS: action + owner + date
KEY QUOTES: exact quotes from transcript

## Template 3: Cold Call Note
[Standard Header Block]
CALL DETAILS: date, outcome, duration
CONVERSATION SUMMARY: 2-3 sentences
PAIN POINTS MENTIONED: their words
OBJECTIONS: objection + handling
NEXT STEP: what was agreed + date

## Template 4: Follow-Up Email Note
[Standard Header Block]
EMAIL DETAILS: date, subject, type
KEY POINTS: main points
RESPONSE: replied Y/N + summary
NEXT ACTION: what + date

## Template 5: Proposal Note
[Standard Header Block]
OPTIONS PRESENTED: name + price + included
THEIR REACTION: interest, concerns, timeline
DECISION MAKER: who decides
BLOCKERS: what's stopping them
TIMELINE: decision timeline
NEXT STEPS: action + owner + date
