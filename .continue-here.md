<decisions_made>

- **8 agents, not 64** — Ruflo has 64 agents but users say "nothing happens." We built 9 purposeful agents with real boundaries.
- **No CRM/BrainOps/Analyst agents** — these are scripts, not agents. Deterministic tasks don't need reasoning.
- **Sonnet for all verification** — Oliver is on Max plan, no token cost. Full quality review on every agent output.
- **Delta-level measurement** — human judgment captured automatically via file edit detection, not manual logging.
- **Cost-tier routing baked into agent definitions** — Haiku for research/scoring, Sonnet for creative/verification. Not a separate routing layer.
- **3-scope memory** — project (shareable patterns), local (deployment-specific), user (personal, never shared). Designed for SDK marketplace.
- **FTS5 over HNSW** — Daem0n uses Qdrant. We use SQLite FTS5 (zero new dependencies) + existing ChromaDB vectors. Hybrid search via RRF.
- **Overnight review at session end** — Oliver's computer is off overnight. brain-maintain.js Stop hook runs review instead of scheduled task.

</decisions_made>
