# Demo Prep

## Gate
Load and enforce domain/gates/demo-prep.md before building any cheat sheet.

## What This Does
Builds a comprehensive demo prep brief with 15-step research, ad platform audit, Story-Trap thread flow design, objection prep, and close strategy. Ensures Oliver walks into every demo with full context.

## Inputs
- Prospect name or company (required)
- Demo date/time (optional — checked via calendar)

## Steps
1. Run the 15-step demo-prep gate checklist (win story, calendar, web, LinkedIn, Apollo, Fireflies, Gmail, NotebookLM, ad audit, thread flow)
2. Present pre-flight proof block and research checkpoint
3. On approval, build the full demo prep brief (account snapshot, thread flow, discovery traps, objection table, close strategy)
4. Draft as HTML email to Oliver
5. Run 5-Point Verification Stack from .claude/gates.md
6. Present to Oliver

## Output
Demo prep brief with Story-Trap structure, ad platform audit, and thread flow design. Saved to docs/Demo Prep/.
