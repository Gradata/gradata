## DOMAIN sections to generate:

### Prospect Research Order
0. CRM — source of truth
1. Vault — brain/ notes (free, instant)
2. Free web — search, LinkedIn, company site
3. Paid enrichment — [tool from wizard] (ask before spending)
Cost rule: exhaust free tools before paid. State cost + ask permission.

### Email Frameworks
- Cold: personalized opener, pain, proof, CTA
- Inbound: welcome, acknowledge, pain, solution, CTA
- Follow-up: their words first, then next step

### CRM Auto-Sync
- Demo booked → create/update deal + pre-demo note
- Demo completed → transcript + notes + stage update
- Follow-up drafted → log email activity
- Always schedule next activity on every deal

### Pipeline Cadences
Post-Demo: day 3, 5, 10, 14
Cold: day 0, 3, 6, 8, 12, 16(breakup)
Re-engage: day 7, 14, 30

### Gates to generate:
1. Pre-Draft Gate (research before any email)
2. Demo Prep Gate (research before any call/demo)
3. Post-Demo Follow-Up Gate (transcript required)
4. CRM Notes Gate (verified data only)
5. Calendar Check Gate (check 2 months out)
