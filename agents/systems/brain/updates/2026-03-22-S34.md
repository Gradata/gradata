# Session 34 Distillation -- Systems Agent

## v2.0 Architecture Changes
- Neural bus replaced by events.jsonl + SQLite events table. Use `from events import emit, query` for all event operations.
- Cross-wires killed entirely. Connections are now hooks + event queries.
- Self-improvement: Instinct (0.0-0.59) -> Pattern (0.60-0.89) -> Rule (0.90+). See .claude/self-improvement.md.
- wrap_up.py now uses update_confidence() instead of decrement_provisionals(). Step-level event logging via log_step().
- 8 new hooks in settings.json: suggest-compact, cost-tracking, session-persist, quality-gate, mcp-health, context-budget, config-validate, secret-scan.
